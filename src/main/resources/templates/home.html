CoinTrends

CoinTrends is an application which verifies whether growths or drops of BITCOIN currency value in USD were preceded growths or drops of any other cryptocurrencies values in USD.

Application allows comparison of historical data of:

  • BTCUSD (bitcoin in USD),
  • ETHUSD (Ethereum in USD),
  • LTCUSD (Litcoin in USD)

In addition, the chart displays the trend line for the listed cryptocurrencies:

  • weekly growth trends are marked by the blue line on the chart,
  • weekly drop trends are marked by the red line on the chart.

Author: Błażej Wohs

How to use:

  1. Write (or pick) start date time
  2. Write (or pick) end date time
  3. Click "Update chart" button
  4. Wait for the chart to load
